# Install MiniKube

*Install the VirtualBox*

`sudo apt-get install virtualbox`

*Install Minikube*

`curl -Lo minikube https://storage.googleapis.com/minikube/releases/v0.25.0/minikube-linux-amd64 && chmod +x minikube && sudo mv minikube /usr/local/bin/`

*Start Minikube*

`minikube start`

*Chech the status*

`minikube status`

*Stop minikube*

`minikube stop`
